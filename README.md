Morse Code Introduction
Morse code is a communication system that uses dots (.) and dashes (-) to send messages. 
Each letter and number has its own pattern. Morse code was used before modern technology to communicate over long distances.
History of Morse Code
Morse code was created by Samuel Morse and Alfred Vail in the 1830s. 
It was developed to work with the telegraph machine, which could send electrical signals through wires. 
Morse code became important because it helped people send messages faster, especially during emergencies and wars.

Messages can be sent using sound, light, or radio signals. A short signal represents a dot, while a longer signal represents a dash.

Conclusion

Morse code played an important role in communication history. Although modern technology is now commonly used, Morse code is still useful in emergencies and radio communication today.
